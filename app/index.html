What are the different types in in javascript?

  • Boolean true or false

  • Number 1,1.0

  • String 'a', "a"

  • Null null

  • Undefined undefined

  • Object {}, new Object()


What does the function typeof() do?

  • The function typeof() can be used to determine what type a value is.

  • typeof(1) the value is "number"

  • typeof('a') the value is "string"

  • typeof(true) the value is "boolean"

  • typeof(undefined) the value is "undefined"

  • typeof(null) the value is "object"

  • typeof({}) the value is "object"


What does it mean that javascript is a dynamically typed language?

  • Javascript variables are determined dynamically at runtime as opposed to languages like java, which expects the variables to be defined ahead of time.


What is the difference between "udnefined" and "null"?

  • When a variable is undefined it has not yet been defined and is set to undefined by the javascript engine, whereas a variable can be assigned to null by the programmer to be manually set undefined.
